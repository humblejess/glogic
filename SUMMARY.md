## 🎯 项目核心

**目标**：通过逻辑思考帮助人们理解性别识别，揭示理性选择与情感反应的不一致性。

**核心流程**：
1. 引言 → 2. 排序互动 → 3. 场景挑战 → 4. 结论 → 5. 行动呼吁

## 🎨 设计方向

### 视觉风格
- **温和而坚定** - 科学、理性的视觉语言
- **包容性设计** - 柔和的色彩，避免二元对立
- **现代简约** - 清晰的信息层级

### 关于人物元素
**推荐方案**：
- ✅ **抽象图形** - 渐变、流动形状，象征连续性
- ✅ **示意性人形** - 极简轮廓、几何组合（如需要）
- ❌ **避免真实照片** - 隐私和敏感性考虑

**设计参考**：
- Notion/Stripe 风格的抽象插画
- CSS 绘制的简单图形
- 等距插画风格（可选）

## 💻 推荐技术方案

### 前端
```
Vue 3 + TypeScript + Tailwind CSS
Vue Router + vue-i18n + vue-draggable-plus
```

### 部署
```
Cloudflare Pages + Cloudflare D1 (SQLite)
Cloudflare Pages Functions (Serverless API)
```

### 理由
- ✅ 项目已有 Vue 基础，快速启动
- ✅ DNS 已在 Cloudflare，部署简单
- ✅ D1 与 Pages 完美集成，无需额外服务
- ✅ 后续可迁移到 Astro 优化性能
